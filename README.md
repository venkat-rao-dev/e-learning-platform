# e-learning-platform

## 🚀 Description

An online learning platform that allows users to access courses, track their progress, and manage their learning journey through a user-friendly interface.

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Styling: Bootstrap

## ✨ Features

* Course listing and access
* User dashboard
* Progress tracking
* Responsive UI design

## ▶️ How to Run

1. Open project in browser
2. Navigate courses and dashboard
3. Explore features
